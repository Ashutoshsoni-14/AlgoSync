# AlgoSync ⚡

AlgoSync is a full-stack **competitive and collaborative coding platform** built for developers, students, and placement preparation. It combines **real-time coding battles** with **live collaborative problem-solving** to create an interactive DSA learning ecosystem.

## 🚀 Features

### ⚔️ Battle Mode

* Real-time coding battles with private editors
* Random problem assignment from curated problem pool
* Shared timer for all players
* First correct submission wins
* Multi-round battle system
* ELO-based ranking updates
* Winner announcements
* Live battle status

### 🤝 Collaborative Mode

* Solve coding problems together in real-time
* Shared code editor with live sync
* Problem selection from curated problem library
* Room chat for discussion
* Best for pair programming and DSA study groups

### 📚 Problem Library

* Based on **NeetCode 150**
* Filter by difficulty:

  * Easy
  * Medium
  * Hard
* Filter by topic:

  * Arrays
  * Hashing
  * Sliding Window
  * Binary Search
  * Trees
  * Graphs
  * Dynamic Programming

### 💻 Code Execution

* Multi-language support:

  * C++
  * Java
  * Python
* Custom code templates
* Run code with custom input
* Submit against test cases
* Instant verdicts

### 📈 Ranking System

* ELO rating system
* Global leaderboard
* Wins / Losses tracking
* Battle history

### 👀 Spectator Mode

* Watch live battles
* View timer and player status
* Track winner in real-time

### 💬 Room Chat

* Real-time messaging inside rooms
* Timestamped chat messages
* Better communication during collaboration

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/your-username/AlgoSync.git
cd AlgoSync
```

### Install frontend dependencies

```bash
cd client
npm install
```

### Install backend dependencies

```bash
cd ../server
npm install
```

### Setup environment variables

Create `.env` inside `server/`

```env
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret_key
```

### Run frontend

```bash
cd client
npm run dev
```

### Run backend

```bash
cd server
npm run dev
```
